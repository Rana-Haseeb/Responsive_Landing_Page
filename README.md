# Nexus Studio

An elegant, responsive landing page for a fictional digital studio. The page blends editorial styling, smooth motion, and a premium dark visual system to present services, pricing, testimonials, and a call to action.

## Features

- Responsive layout for desktop and mobile
- Custom cursor and follower effect on fine-pointer devices
- Scroll-triggered reveal animations
- Animated hero counters
- Pricing toggle for monthly and annual views
- Magnetic hover interactions for service cards
- Smooth scrolling and active navigation states

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts: Cormorant Garamond, DM Mono, Syne

## Project Structure

- `index.html` - page markup and content
- `styles.css` - visual design, layout, and responsiveness
- `script.js` - interactions and animations

## Getting Started

1. Open the project folder in VS Code.
2. Open `index.html` in a browser, or use Live Server if available.
3. Edit the content, styles, or interactions as needed.

## Notes

- The contact email in the page currently uses a placeholder domain for the new brand.
- No build step is required. This is a static front-end project.
